# tococo
Um diário cocô para usuários de terminal no Linux.

Tá meio **cocô** hoje e não entende bem o motivo? Precisa clarear ideias na cabeça?

`tococo` é uma forma rápida de fazer isso. Apenas abra o terminal e bote pra fora :)

Apenas um arquivo por dia. Todos os arquivos são salvos no formato `ano-mes-dia.txt` em uma pasta (manualmente) configurável.

_Só quis colocar aqui por diversão/recordação. Nada muito sério a ser levado._

## Instalação

Clone este repositório e mova `tococo` para `~/.local/bin/` ou qualquer outro diretório que tenha como PATH.
Edite `tococo` para decidir o local em que serão salvos os arquivos e também seu editor de texto (default: `micro`).
